## Setup

*Don't forget to install packages in both projects.*<br>
*You need yarn to be installed on your machine.* 
```

npm i

cd

cd server
yarn install
```

### 1. Run SPA
Go to the project folder:
```
cd q4
```
Run build:
```
npm run start
```

### 2. Start a test server
Go to the project folder:
```
cd server
```
```
yarn run server
```
