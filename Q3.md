Static HTML page with responsive grid. Project styles are being compiled with
 SCSS pre-processor. No Bootstrap or other vendor library used.

## Setup

Go to the project folder
```
cd q3
```

Install packages
```
npm i
```

Generate CSS
```
npm run sass
```

---

### Development

Watch and re-compile CSS each time you save SASS:
```
npm run sass-w
```
